Install Compiler

There are two different compilers that we must install. The Protobuf has a main compiler that can compile .proto files into Python, Go, Rust, C#, Java and C++. Once installed its command is protoc.

In order to compile into JavaScript output, we have to install another distribution of the compiler which is in the form of a Node.js package. Once installed its command is pbjs.

Install Main Compiler​

  • Windows
  • Linux
winget install protobuf
protoc --version
Alternative way to install the compiler

Download the distribution matching your operating system from the compiler's repo and put it in path.

apt install protobuf-compiler -y
protoc --version
Alternative way to install the compiler

Download the distribution matching your operating system from the compiler's repo and put it in path.

cd ~
wget https://github.com/protocolbuffers/protobuf/releases/download/v3.20.1/protoc-3.20.1-linux-x86_64.zip
unzip protoc-3.20.1-linux-x86_64.zip -d protoc
sudo chmod a+x ~/protoc/bin/protoc
nano .profile
	export PATH="$PATH:$HOME/protoc/bin/"
source .profile

Install JavaScript-Specific Compiler​

npm install protobufjs -g
pbjs

Now we have a command name protoc in our command line interface. For example we can check the version of the compiler with the protoc --version command.
