# Credentials

Once we are connected to the server, we can send a message to it,  
and the server will respond to our message too, but the problem is  
because server does not know who we are, it will respond to any message  
we send to it by sending an error message back. In order to introduce ourself  
properly to the server, we need to have the proper credentials.

In this section, we cover how to get the proper credentials and more.
